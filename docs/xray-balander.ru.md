# Обновлятор балансера: _xray-balancer.py_

У меня есть _несколько_ VPS-ок с иксреем, и все их надо упихать в один клиент на одомашненном сервере
omelab

## Установка
1. Куда-нибудь скачиваем  
2. Chmod-аем его по самые +x пермы  
3. Пишем базовый конфиг  
4. Кроним скрипт
